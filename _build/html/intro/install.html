快速安装指南¶

Before you can use Django, you’ll need to get it installed. We have a complete installation guide that covers all the possibilities; this guide will guide you to a simple, minimal installation that’ll work while you walk through the introduction.

安装Python¶

Being a Python Web framework, Django requires Python. See 我应该使用哪个版本的Python来配合Django? for details. Python includes a lightweight database called SQLite so you won’t need to set up a database just yet.

Get the latest version of Python at https://www.python.org/download/ or with your operating system’s package manager.

Django on Jython

如果你使用 Jython (一个Java平台的Python实现), 你将需要进行一些额外的步骤. 请查看 Running Django on Jython 以获得更多细节.

你可以在你的shell中输入 python 来确定你是否安装过Python; 你看到的可能是像这样子的:

Python 3.4.x
[GCC 4.x] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

设定一个数据库¶

此步骤仅在你打算使用诸如 PostgreSQL, MySQL, 或者 Oracle这些大型数据库引擎时需要. 要安装这样一个数据库, 请参考 数据库安装信息.

移除任何老版本的Django¶

如果你打算从前一版本的Django升级，, 你将需要 在安装新版本前删除旧版本的Django.

安装Django¶

你有三个简单的方法来安装Django:

  • 安装 django 操作系统所提供的发行包 版本. 对于操作系统提供了 Django安装包 的人来说，这是最快捷的安装方法。

  • 安装官方正式发布的版本. 这对于想要安装一个稳定版本而不介意运行一个稍旧版本的Django的用户来说是最好的方式。

  • 安装最新开发版. 这个选择是针对那些想要最新和最好的特性的爱好者们，并不怕运行全新代码. 你在开发版中可能会遇到新的错误BUG, 但可以报告给社区团队帮助 Django开发. 此外, 第三方发行的软件包也可能不与开发版进行兼容。

请始终参考与你所使用的版本对应的 Django文档！

如果采用了前两种方式进行安装, 你需要注意在文档中标明 在开发版中新增. 这个标记表明这个特性仅适用开发版 Django, 并且他们可能不会在官方发布的稳定版中工作。

验证¶

若要验证 Django可以被 Python 识别, 可以在shell中输入 python . 然后在 Python 提示符下，尝试导入 Django：:

>>> import django
>>> print(django.get_version())
1.10

当然了，你也可能安装的是其它版本的 Django。

就是这个！¶

That’s it – you can now move onto the tutorial.
